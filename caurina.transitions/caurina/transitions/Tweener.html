Classpublic class Tweener
InheritanceTweener Object

Tweener is a static class that allows attaching property transitions to objects. The class supports built-in display objects, as well as custom objects.

Getting started

To create a tween, simply call Tweener.addTween(), as in the following snippet:

     // Slide a movie clip to a new position in 0.5 seconds
     Tweener.addTween(myMovieClip, { x: 10, time: 0.5 });
     
     // Fade in on a movie clip in 2 seconds
     myMovieClip.alpha = 0;
     Tweener.addTween(myMovieClip, { alpha: 1, time: 2 });
     
     // Do a transition on both x and alpha
     Tweener.addTween(myMovieClip, { alpha: 1, x: 10, time: 0.5 });

Refer to the Tweener.addTween method documentation for more details.

Events

Tweener.addTween takes optional event listeners such as onStart, onUpdate and onComplete:

     function go(): void { trace("I'll go!"); }
     function going(): void { trace("I'm going!"); }
     function gone(): void { trace("I'm gone!"); }
     
     Tweener.addTween(myMovieClip, { alpha: 0, time: 1, onStart: go, onUpdate: going, onComplete: gone });

addCaller()method
public static function addCaller(scopes:Object = null, options:Object = null):Boolean

Adds a new caller tweening.

Options object

         type Options = {
                // Time in seconds or frames for the tweening to take. Defaults to 2.
             time?: Number,  
             // Delay time. Defaults to 0.
             delay?: Number,
             // Number of times this caller should be called.
             count?: Number,
             // Whether to wait (or not) one frame for each call.
             waitFrames?: Boolean,
             // A transition equation either as one of the constants defined by the `Easing` class
             // or an `EquationFunction` function.
             // Defaults to Easing.OUT_EXPONENTIAL.
             transition?: String | Function,
             // Whether to use rounded values when updating or not.
             rouunded?: Boolean,
             // How many updates should be skipped. Defaults to 0.
             skipUpdates?: Number,
         
             onStart?: (...arguments: Array) => void,
             onUpdate?: (...arguments: Array) => void,
             onComplete?: (...arguments: Array) => void,
         };
         
         // Function that returns the correct value.
         //
         // - `t` is the current time (in frames or seconds)
         // - `b` is the starting value.
         // - `c` is the change needed in value.
         // - `d` is the expected easing duration (in frames or seconds).
         // - `params` are additional parameters to the function.
         //
         type EquationFunction = (t: Number, b: Number, c: Number, d: Number, params?: Object) => Number;

Parameters

scopes:Object (default = null) — Object that should be tweened: a MovieClip, TextField, a custom object, or an Array of objects.
 
options:Object (default = null) — Object containing the specified parameters in any order, as well as the properties that should be tweened and their values.

Returns
Boolean — True if the tween was successfully added, false if otherwise
